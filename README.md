Retail Sales Analysis Dashboard



Project Overview:

  This project focuses on analyzing retail sales data to extract meaningful insights and visualize key business metrics using Power BI.
  The goal is to understand sales performance, customer behavior, and product trends to support data-driven decision-making.


Objectives:

  1)Analyze sales data to identify trends and patterns.
  2)Track key performance indicators (KPIs) such as total sales, average MRP, and outlet performance.
  3)Provide actionable insights for business improvement.
  4)Build an interactive dashboard for better visualization.

Tools & Technologies:

  1)Python (Pandas, NumPy)
    2)SQL
3)Power BI


Process:

  1)Data Cleaning:
  Handled missing values
  Removed duplicates
  Ensured data consistency

  2)Data Analysis:
  Analyzed sales by outlet type, location, and item category
  Identified top-performing products and regions
  
  3)Data Visualization:
  Created interactive Power BI dashboard
  Displayed KPIs such as total sales, number of items, and average MRP


Key Insights:

  Tier 3 outlets generated the highest sales
  Certain product categories contributed significantly to total revenue
  Sales trends varied based on outlet size and location



Dashboard Features:
  KPI cards (Total Sales, Average MRP, Total Items)
  Sales by outlet type and location
  Item category analysis
  Year-wise outlet establishment trends


Conclusion

  The project demonstrates how data analysis and visualization can be used to uncover valuable business insights and improve decision-making.
  It highlights the importance of data-driven strategies in retail performance optimization.
